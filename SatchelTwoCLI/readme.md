## The bare minimum for checking your homework!

### So... What is this?

So yeah, this is the original Python scripts that do all the fancy stuff with converting SatchelOne's ICAL files to CSV's in a nice clean format. And yes, I do understand I've basically made SatchelOne for Arch Users. You're welcome.

### Requirements

Electricity, Python 3.9 or higher (I think) and Copious ammounts of sanity

### Is this still going to be updated along with the GUI version?

Uhh... I hope so? Depends if I have the time on my hands or if I want to check my homework on any device that runs Python 3. If something breaks I'll fix it obviously, but don't expect much.
